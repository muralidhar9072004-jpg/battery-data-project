# Battery Data Project 

Python project for analyzing battery station performance data.

## Features
- Average SOC calculation
- Detect low battery stations
- Identify weak battery health
- Export report to CSV

## Tech Stack
- Python
- Pandas

  ## Sample Insights

- Average SOC: 65.4%
- Low SOC Stations: BLR02, BLR05
- Weak Health Batteries: BLR02, BLR05
- Maintenance Flagged: BLR04

- battery_analysis.py
- battery_report.csv
